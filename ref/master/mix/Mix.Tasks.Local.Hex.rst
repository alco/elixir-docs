Overview
--------

Install hex locally from https://hex.pm/installs/hex.ez.

::

    mix local.hex

Command line options
~~~~~~~~~~~~~~~~~~~~

-  ``--force`` - forces installation without a shell prompt. Primarily
   intended for automation in build systems like make.
