Modules
-------

================================= =
:elixir:mod:`ExUnit`              Basic unit testing framework for Elixir 

:elixir:mod:`ExUnit.Assertions`   This module contains a set of assertion functions that are imported by default into your test cases 

:elixir:mod:`ExUnit.Callbacks`    Defines ExUnit Callbacks 

:elixir:mod:`ExUnit.CaptureIO`    Functionality to capture IO for testing 

:elixir:mod:`ExUnit.Case`         Sets up an ExUnit test case 

:elixir:mod:`ExUnit.CaseTemplate` This module allows a developer to define a test case template to be used throughout his tests. This is useful when there are a set of functions that should be shared between tests or a set of setup/teardown callbacks 

:elixir:mod:`ExUnit.DocTest`      ExUnit.DocTest implements functionality similar to `Python's doctest <http://docs.python.org/2/library/doctest.html>`__ 

:elixir:mod:`ExUnit.Filters`      Conveniences for parsing and evaluating filters 

:elixir:mod:`ExUnit.Formatter`    This module holds helper functions related to formatting and contains documentation about the formatting protocol 

:elixir:mod:`ExUnit.Test`         A struct that keeps information about the test 

:elixir:mod:`ExUnit.TestCase`     A struct that keeps information about the test case
